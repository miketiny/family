# 🏠 电子族谱 Family Tree

> 一个简单易用的电子族谱系统，轻松记录家族血脉传承。

---

## ✨ 功能特性

| 特性 | 说明 |
|------|------|
| 📋 族谱列表 | 浏览所有已创建的族谱 |
| 🔐 密码保护 | 输入家族密码即可查看族谱 |
| 🔗 私密链接 | 专属私密链接，无需重复输入密码 |
| 📊 竖向查看 | 支持竖向树形图展示 |
| 📝 历史版本 | 查看族谱编辑历史记录 |

## 🚀 快速开始

1. 将项目部署到您的 PHP 服务器
2. 访问首页，点击「创建我的族谱」
3. 输入姓氏和四位密码，生成专属链接
4. 分享链接给家人，共同完善族谱信息
